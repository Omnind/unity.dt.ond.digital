{"Title":"Indicator Light","Text":"<size=16><b>Indicator Light</b></size>\r\n\r\nTo add an <b>Indicator Light</b> to a GameObject, select the GameObject you want to asign it to in the hierarchy. Then go to <i>Standard Components > Indicator Light > Add Physical Component</i>.\r\n\r\n<b>Important:</b> An Indicator Light can only be added to a GameObject if it already has a <b>MeshFilter</b> and a <b>MeshRenderer</b> component.\r\n\r\nUnder the tab <i>Live Data</i> tab you can read the live data of the component. Here you can see its intensity is and whether it is currently active.\r\n\r\nIn the <i>properties</i> tab you can select two colors:\r\n- <b>Light Color</b>: This is the color the Indicator Light emits when set to active.\r\n- <b>Base Color</b>: This is the 'inactive', non-emitting color of the Indicator Light.\r\n\r\nHere you also need to <b>select the Material</b> you want to actually be lit up as the light.\r\n\r\nTo be able to control the component, create a <b>Control Panel</b> for the component from the <i>Control Panel</i> tab.\r","DocumentationURL":"https://unit040.atlassian.net/wiki/spaces/PUD/pages/280953809/Indicator+Light","SubComponents":"","Locked":true}
